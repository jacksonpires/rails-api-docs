POST /users

Create a User

Register a new user account in the system. Returns the newly created user record along with an authentication token that can be used for subsequent requests.

Body

name string Required example
Full name of the user. Will be used as the display name across the app.
email string Required example
A unique email address. Must be a valid RFC 5322 address. Used for sign-in.
password string write-only Required min: 8 example
Plain-text password. Hashed server-side with bcrypt before storage. Never returned in responses.
birthdate date example
ISO 8601 formatted date (YYYY-MM-DD). Optional.

Responses

▶ 201 Created — user successfully created
▶ 400 Bad Request
▶ 401 Unauthorized
▶ 409 Conflict — email already in use
▶ 422 Unprocessable Entity
