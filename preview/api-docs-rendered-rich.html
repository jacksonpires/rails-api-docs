POST /users 🔒 Bearer auth

Create User

Register a new user account.

Headers

Authorization string Required
Bearer token from /sessions endpoint
Example: Bearer eyJhbGciOiJIUzI1NiIs...
X-Idempotency-Key string
Unique key to make this request idempotent.

Body

name string Required
Full name of the user.
Example: Marc Andersson
email string email Required
Unique email.
Example: marc@example.com
password string write-only Required min: 8 max: 72
Plain-text password. Hashed server-side.
role string default: user
one of: user · admin · guest
User's permission level.
age integer nullable min: 0 max: 150
zip string pattern: ^\d{5}$
5-digit US ZIP.

Responses

201 User created successfully

Headers

Location string
Canonical URL of the created resource.
Example: https://api.example.com/users/42

Schema

id integer read-only
Example: 42
name string
Example: Marc Andersson
email string email
token string read-only
JWT auth token, valid for 7 days.
409 Conflict — email already in use
422 Validation failed

Schema

errors object
Map of field name → array of validation messages.
